# 风格规则

## 总体风格

文章应该是：

```text
清晰
克制
有锋芒
有推导链
有边界意识
```

不要写成：

```text
学术论文
情绪化檄文
立场宣言
鸡汤文章
术语堆砌
```

## 必须做到

### 1. 把道德问题改写成结构问题

不要只写：

```text
年轻人不负责任。
男女越来越极端。
社会越来越割裂。
```

要改写成：

```text
什么结构条件让某些群体形成这样的行为或观念？
什么制度、利益、成长经验、评价体系塑造了这种结果？
```

### 2. 把大词拆成变量

如果使用这些词，必须解释具体含义：

```text
结构
原子化
现代性
父权
女权
资本
阶层
主体性
意识形态
异化
宏大叙事
```

不要让这些词直接充当解释。

错误示例：

```text
这是资本导致的。
这是现代性的结果。
这是父权制的问题。
```

更好的写法：

```text
这里的"资本"具体指平台以绩效指标和劳动替代性来压低个体议价能力。
它通过 A、B、C 三个机制影响结果。
```

### 3. 每个重要结论都要有推导链

不要直接下结论。

使用：

```text
P1 + P2
→ 中间机制
→ 结果
```

### 4. 不要群体本质化

不要写：

```text
男人就是……
女人就是……
年轻人都……
老一辈都……
底层人都……
```

应该写：

```text
在某些结构条件下，一部分群体更容易形成……
这种倾向不能推出所有个体都如此。
```

### 5. 区分事实判断和价值判断

事实判断回答：

```text
发生了什么？
为什么发生？
通过什么机制发生？
```

价值判断回答：

```text
这是否合理？
是否应该被支持？
评价标准是什么？
```

不要把价值判断伪装成事实判断。

## 推荐表达

可以使用：

```text
这个问题表面上是 A，实质上是 B。
这不是单一原因导致的，而是多个机制叠加的结果。
这个结论只在某些边界内成立。
这套解释不是为了替任何一方辩护，而是为了说明冲突如何生成。
真正的分歧不在结论，而在前提。
```

## 禁止倾向

避免：

```text
过度嘲讽
道德审判
单因解释
用标签替代分析
把复杂问题写成阵营对抗
用"很明显""本质上就是"掩盖推导缺失
```
