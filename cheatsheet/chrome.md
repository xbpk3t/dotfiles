## webpage操作

- `CMD + +/-`：放大/缩小页面
- `CMD + 0`：重置页面缩放比例

---

- `CMD + R`：**深度刷新**
- `CMD + Shift + R`：强制重新加载（清除缓存）


- `空格键`：**向下滚动网页**
- `Shift + 空格键`：向上滚动整屏

- `CMD + ↑`：滚动到页面顶部
- `CMD + ↓`：滚动到页面底部


---

- `Fn + ←`：滚动到顶部（= Home键）
- `Fn + →`：滚动到底部（= End键）
- `Fn + ↑`：向上翻页（= Page Up）
- `Fn + ↓`：向下翻页（= Page Down）
- `Fn + CMD + F`：全屏切换

---

- `CMD + d` # 保存到书签



## 🪟 标签页和窗口快捷键

- `CMD + T`：**新建标签页**
- `CMD + Shift + T`：**重新打开最近关闭的标签页**（支持连续恢复多次关闭）

---

- `CMD + N`：**新建窗口**
- `CMD + Shift + N`：新建隐身窗口（隐私浏览）

---

- `CMD + Shift + [/]`：**切换tab** # 相当于 `CMD + Option + ←/→`
- `CMD + W`：关闭当前标签页


## 🔍 地址栏快捷键

- `CMD + L`：**聚焦到地址栏**（可直接编辑网址或搜索）
- `CMD + Option + F`：在页面内查找文本（激活搜索框）

- ~~`CMD + Enter`~~：**在新标签页执行搜索/打开网址**（保持当前页面活动状态） # “同样，搜索后不跳转到新标签页。在有大量搜索的场景下，效率很高，少了跳转和重复键入搜索词条的过程。”


## 链接操作

- `CMD + 点击URL`：**后台新标签页打开** # “在新 tab 打开该 url，不需要再右键操作了，操作效率更高”
- `CMD + Shift + 点URL` # 在【新tab】打开该 url，并切换到【新tab】
- `Shift + 点击URL`：**新窗口打开并立即聚焦**
- 鼠标中键点击链接：等同于 `CMD + 点击`（后台打开）



## chrome toolbar操作

- `CMD + Option + B`：**打开书签管理器**
- `CMD + Shift + J`：**打开下载内容**
- `CMD + Y`：**打开历史记录**




## ⚡ 高效操作组合键

- `CMD + Option + I`：打开开发者工具
- `CMD + ,`：打开设置页面
- `CMD + H`：隐藏Chrome窗口
- `CMD + Q`：完全退出Chrome
